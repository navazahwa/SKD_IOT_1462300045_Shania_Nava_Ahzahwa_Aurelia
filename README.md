# Tugas IoT: Sistem Deteksi Gerakan dengan Arduino

## Deskripsi
Simulasi sistem lampu otomatis menggunakan
Arduino UNO
Sensor PIR HC-SR501
LED

## Cara Kerja:
Sensor PIR mendeteksi gerakan.
Arduino menyalakan LED via Pin 13 selama 5 detik.
